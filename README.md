# 話者分離プロジェクト

このプロジェクトは、音声データから複数の話者を分離するための実装です。

## 機能

- 音声ファイルからの話者分離
- 話者ごとの音声セグメンテーション
- 分離された音声の可視化

## ディレクトリ構造

```
.
├── data/          # 音声データファイル
└── docs/          # ドキュメント
```

## セットアップ

1. リポジトリのクローン:
```bash
git clone [your-repository-url]
cd speaker-diarization
```

2. 必要なパッケージのインストール:
```bash
pip install -r requirements.txt
```

詳細な使用方法は `docs/` ディレクトリ内のドキュメントを参照してください。
